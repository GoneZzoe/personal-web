# HaoLi - Personal Portfolio Website

This is a clean, modern, single-page personal portfolio website built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**. 

## 🚀 Features

- **Single-Page Application (SPA)**: Clean, streamlined scrolling experience mapping all sections on the homepage.
- **Smooth Navigation**: Custom header and mobile dropdown links that scroll smoothly to respective targets.
- **Scroll Spy**: Auto-highlighting navbar indicators that track active page sections using the `IntersectionObserver` API.
- **Resume Section**: A collapsible PDF resume previewer utilizing an iframe element, accompanied by a download option.
- **Work Experience & Projects**: Detailed timeline cards rendering achievements and projects with structured bullet-point formatting.
- **SEO Optimized**: Fully integrated metadata, descriptions, and tag configurations.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS configurations
- **Components**: shadcn/ui & Radix UI primitives
- **Utility**: Lucide/Tabler Icons, dayjs, clsx

---

## 💻 Getting Started

First, install dependencies:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Building for Production

To generate an optimized production bundle:

```bash
npm run build
```

The build output will be compiled and validated locally under static pages output.
